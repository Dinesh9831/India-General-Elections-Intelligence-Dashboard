
# India-General-Elections-Intelligence-Dashboard 🇮🇳

## 📊 Project Overview
A comprehensive Power BI dashboard designed to analyze Indian General Elections data at national, state, constituency, and party levels.  
This project transforms raw election data into meaningful insights through interactive and visually rich analytics.

---

## 🚀 Key Features

### 🔹 Executive Overview
- Total Constituencies, Candidates, Parties  
- Total Votes & Voter Turnout %  
- National-level vote distribution  

### 🔹 State Analysis
- State-wise voter participation  
- Candidate density across states  
- Regional voting patterns  

### 🔹 Constituency Analysis
- Winner, Runner-up & Vote Margin  
- Candidate vote share  
- Gender participation  

### 🔹 Party Strategy & Performance
- Party-wise vote share  
- Top performing parties  
- Competitive analysis  

### 🔹 Advanced Analytics
- Decomposition Tree (Vote Drivers)  
- Key Influencers (Factors affecting votes)  
- Scatter & Area charts  

---

## 🛠️ Tech Stack
- Power BI  
- DAX (Data Analysis Expressions)  
- Power Query (M Language)  
- Data Modeling  

---

## 📁 Dataset Columns
- `st_name` – State Name  
- `year` – Election Year  
- `pc_name` – Constituency Name  
- `cand_name` – Candidate Name  
- `cand_sex` – Gender  
- `partyname` – Party Name  
- `partyabbre` – Party Abbreviation  
- `totvotpoll` – Total Votes Polled  
- `electors` – Total Electors  
- `pc_type` – Constituency Type (GEN/SC/ST)
